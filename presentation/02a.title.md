# You Might Not Need NixOS

![NixOS Logo](./attachments/nixos.png)

* NixOS is an extremely-hyped distribution of Linux.
* It claims to offer many advantages over other systems.
* But I'm pretty sure there are alternatives that can do everything it does, but
  better.
* And so I'm going to try to convince myself (and all you lovely viewers) of 
  this fact.
